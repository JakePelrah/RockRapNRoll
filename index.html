A brief history of Rock Rap N' Roll

By Jake Pelrah

Rock Rap N' Roll is a music education game authored by Barry Schuler, Nick Tenbrock and Gary Levenberg at Medior Inc, and published by Paramount Interactive. Released in 1993 RRR allows the user to construct songs in various styles, using studio produced music clips. Mega Rock Rap N' Roll demonstrated below, is an extension of RRR that included more music genres. Convolution reverb was added using the Web Audio ConvolverNode. Impulse responses used in this project can be found at Fokke van Sanne.
